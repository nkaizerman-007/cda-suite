# CDA Intelligence Suite

A two-product AI platform built for Content Discovery Agency —
Israel's first content discovery agency.

## Live Site
[View minisite](./cda-v5.html)

## Three Repositories

| Project | What it is | Repo | Access |
|---------|-----------|------|--------|
| **CDA Minisite** | Marketing landing page for both products | (this repo) | Public |
| **CDA Web — Media Matching Platform** | Full-stack AI matching engine — React, FastAPI, Supabase, Claude API, 8-agent pipeline, 4 permission levels | [CDA_web](https://github.com/nkaizerman-007/CDA_web) | Private |
| **Sherlock · Deal Intelligence** | Signal-based ABM targeting — CrewAI, 8 agents, Streamlit dashboard, ML model (Logistic Regression, Random Forest, XGBoost), full data analysis pipeline (EDA, feature engineering, model evaluation) | [Sherlock](https://github.com/nkaizerman-007/Sherlock) | Private |
